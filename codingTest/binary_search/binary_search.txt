이진 탐색이란?
 "반으로 쪼개면서 탐색하기"
 "찾으려는 데이터와 중간점 위치에 있는 데이터를 반복적으로 비교" 
 무작위로 배치된 배열을 시작점, 끝점, 중간점을 활용하여 원하는 값을 찾는 알고리즘이다.
 
 과정
  1)데이터 정렬
  2)시작점,끝점을 설정한다.
  3)시작점 과 끝점 사이의 중간값(몫 평균)을 구한다(단 소수점 이하의 자릿수는 제외(몫만 사용))
  4)중간점 값과 찾는 데이터를 비교한다
  5)if 중간점 값 > 찾는 데이터 : 끝점을 중간점 앞 인덱스를 가리키게 함
    elif 중간점 값 < 찾는 데이터 : 시작점을 중간점 뒤의 인덱스를 가리키게 함
    elif 준간점 == 데이터 : 종료
  6) 값을 찾기까지  다시 3번과정부터 반복한다.

 재귀함수로 구현하는것이 효울적이다.
 (반복문을 활용하여 구현도 할수 있다.)
시간복잡도
O(logN)
*** 탐색범위가 1,000만 단위가 넘어가면 이진탐색과 같이 O(logN)의 속도로 알고리즘을 해결해야하기 때문에
     중요한 알고리즘 이다!!

##인진 탐색 알고리즘을 필요로하는 문제들 특징
 이진 탐색을 필요로 하는 문제들은 입력 데이터가 많거나, 탐색범위가 매우 넓은 경우에 주로 사용한다.
 그렇기 때문에 input()함수를 사용해서 시간 초가 판정을 받을 수 있기 때문에 sys 라이브러리를 사용하자!.

sys 라이브러리 readline()
	import sys
	#하나의 문자열 데이터 입력받기
	#rstrip() \n기호를 제거하기 위한 코드 반드시 입력하기!!
